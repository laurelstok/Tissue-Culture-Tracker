GitHub workflow
  1. After each session click Download JSON
  2. In VS Code drag the file into your Tissue-Culture-Tracker folder, replacing the old one
  3. In the terminal: git add tc-records.json && git commit -m "update records" && git push
  4. Next session: run python3 -m http.server 8080, open http://localhost:8080, then import your JSON
